# Improvement Ladder

A personal daily courage app — three small social-courage "dares" a day, drawn from a tiered ladder, with streaks, points, levels, a morning quote, an optional evening journal, and a history view.

Single-page offline PWA. All data lives in the browser's localStorage on the device — no accounts, no server, no analytics, no network calls at runtime.

## Use it on your phone
1. Open the site in Safari (iPhone) or Chrome (Android).
2. Add to Home Screen.
3. Open it once from the icon — it caches for full offline use.
